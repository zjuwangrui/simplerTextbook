# 接口文档

后端基地址：`/api`

## 健康检查

### `GET /health`

返回服务状态、教材数量和 LLM 是否启用。

## 教材管理

### `GET /textbooks`

返回教材列表。

### `POST /textbooks/upload`

`multipart/form-data`

- 字段名：`files`
- 支持：`pdf`、`docx`、`txt`、`md`

返回 `202 Accepted`，并立即写入教材占位记录。后端随后在后台解析文件。

返回字段包括：

- `id`
- `filename`
- `title`
- `format`
- `file_size_bytes`
- `status`
- `parse_progress`

### `GET /textbooks/{textbook_id}`

返回教材完整详情，包括：

- 基础元信息
- 统一教材结构：`textbook_id / filename / title / total_pages / total_chars / chapters`
- `parsed_output`：与上传文档要求对齐的标准化结构
- 章节切分结果
- 检索块
- 关键词
- 单教材知识图谱
- 摘要预览

### `GET /textbooks/{textbook_id}/status`

返回后台解析状态：

- `status`
- `parse_progress`
- `error_message`
- `total_pages`
- `total_chars`
- `detail_url`

说明：

- `/status` 只用于进度轮询，不返回完整章节树。
- 解析完成后，应调用 `/api/textbooks/{textbook_id}` 获取完整教材结构。

## 图谱

### `GET /graphs/textbooks/{textbook_id}`

返回指定教材的知识图谱：

- `nodes`
- `edges`
- `stats`
- `relation_types`
- `chapter_graphs`
- `description`

### `POST /graphs/textbooks/{textbook_id}/generate`

返回 `202 Accepted`，异步触发单本教材知识图谱生成。

### `GET /graphs/textbooks/{textbook_id}/status`

返回图谱生成状态：

- `graph_status`
- `graph_progress`
- `graph_error_message`

说明：

- 该接口返回的是章节级 LLM 抽取后的知识图谱可视化 JSON。
- `nodes` 中包含 `id / name / definition / category / chapter / page`。
- `edges` 中包含 `source / target / relation_type / description`。

### `POST /graphs/combined`

请求体：

```json
{
  "textbook_ids": ["book_a", "book_b"]
}
```

返回多教材组合图谱。

## 跨教材分析

### `POST /analysis/compare`

请求体：

```json
{
  "textbook_ids": ["book_a", "book_b"]
}
```

返回：

- `overview`
- `shared_keywords`
- `pairwise_similarity`
- `unique_topics`
- `missing_topics`
- `combined_graph`

## 内容整合

### `POST /integration/generate`

请求体：

```json
{
  "textbook_ids": ["book_a", "book_b"],
  "ratio": 0.3
}
```

返回：

- `summary_ratio`
- `original_characters`
- `summary_characters`
- `summary_text`
- `items`

## RAG 问答

### `POST /qa/ask`

请求体：

```json
{
  "textbook_ids": ["book_a", "book_b"],
  "question": "这些教材对核心概念的共同定义是什么？"
}
```

返回：

- `mode`
- `question`
- `answer`
- `citations`

说明：

- 未启用外部 LLM 时，返回本地检索+摘录式答案。
- 启用后，调用 OpenAI 兼容接口生成带引用回答。

## 教师对话

### `GET /dialogue/history`

返回历史反馈记录。

### `POST /dialogue/message`

请求体：

```json
{
  "textbook_ids": ["book_a", "book_b"],
  "message": "第三章压缩过多，请保留案例。"
}
```

返回：

- `teacher_message`
- `assistant_message`
- `mode`
- `timestamp`

## 报告

### `GET /reports`

返回已生成报告列表。

### `POST /reports/generate`

请求体：

```json
{
  "textbook_ids": ["book_a", "book_b"]
}
```

返回：

- 报告元信息
- 文件保存路径
- Markdown 内容
